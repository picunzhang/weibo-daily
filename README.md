# 每日简报 - 墨水屏版

适配 7.3 寸墨水屏（800x480, 5:3 比例）的微博热搜每日简报。

## 功能

- 显示当前日期和时间
- 显示微博热搜前10条
- 适配 7.3 寸墨水屏（黑白显示优化）
- 每日北京时间 12:00 和 00:10 自动更新

## 技术栈

- Python 数据抓取
- GitHub Actions 定时任务
- GitHub Pages 静态托管

## 部署

1. Fork 此仓库
2. 在 Settings > Pages 中开启 GitHub Pages（选择 main 分支）
3. 定时任务会自动运行，也可以在 Actions 页面手动触发
